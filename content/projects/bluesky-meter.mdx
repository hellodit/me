# Bluesky Meter

A real-time analytics platform for the Bluesky social network, providing insights into account growth, engagement metrics, and content performance.

## Features

- Real-time follower tracking
- Engagement analytics
- Content performance metrics
- Growth trends visualization

## Tech Stack

- Next.js
- TypeScript
- Bluesky API
- Chart.js

## Status

This project was successfully sold in 2024. It served thousands of Bluesky users and provided valuable insights into social media analytics.
