# Analytics Platform

An open-source analytics platform designed to help developers track and understand user behavior on their websites.

## Features

- Real-time analytics
- Custom event tracking
- User session recording
- Privacy-focused design
- Open source

## Tech Stack

- Next.js 16
- TypeScript
- PostgreSQL
- Prisma ORM
- Tailwind CSS

## Current Status

The project is currently in active development. We're working on:

- Core tracking functionality
- Dashboard UI
- API endpoints
- Documentation

Contributions are welcome! Check out the GitHub repository to get started.
