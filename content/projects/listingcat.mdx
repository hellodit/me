# Listing Cat

A comprehensive marketing database that helps startups and indie hackers discover marketing opportunities, directories, and platforms to promote their products.

## Features

- Curated list of marketing platforms
- Search and filter functionality
- Detailed platform information
- Community-driven updates

## Impact

Listing Cat has helped hundreds of startups and indie hackers find the right marketing channels for their products.

## Tech Stack

- Next.js
- React
- Tailwind CSS
- Supabase

## Current Status

The platform is actively maintained and regularly updated with new marketing opportunities.
